package glm

Static methods

static axisAngle (axis :Vec3, angle :Float) :Quat

Constructs a quaternion which represents a rotation of angle around the axis axis.

Parameters:

axis

The axis to rotate about

angle

The angle to rotate by (in radians)

Returns:

A quaternion which can be used to rotate vectors in 3D

static rotate (q :Quat, v :Vec3) :Vec3

Rotates the vector v by the quaternion q (typically constructed using axisAngle). Applies q * v * q^-1

Parameters:

q

The quaternion to rotate with

v

The vector to rotate

Returns:

v, rotated by q

static scale (?m :Mat4, scale :Vec3) :Mat4

Scales a matrix by the 3D scale vector

Parameters:

?m

A matrix to apply the transformation on. If m == null, an identity matrix will be used.

scale

A vector describing the scaling

static translate (?m :Mat4, translation :Vec3) :Mat4
